Student Feedback Management System

A simple web application that allows students to submit course feedback, professors to view analytics, and admins to manage courses.
Built using PHP, MySQL, Bootstrap, and Chart.js.

Features
Student
- Submit course feedback
- Rate overall teaching & materials
- Add comments
- Clean and simple UI

Professor
- View courses assigned to them
- View student comments
- Charts using Chart.js
- See average ratings

Admin
- Add new courses
- View all courses
- Manage deadlines
- Export feedback to CSV

Technologies
- PHP
- MySQL
- HTML/CSS
- Bootstrap 5
- Chart.js

How to Run (Localhost)
1. Install XAMPP
2. Place the project folder in:
   C:\xampp\htdocs\
3. Start Apache and MySQL
4. Create a database in phpMyAdmin (e.g., student_feedback)
5. Import your SQL tables
6. Update /config/db.php with your database details
7. Open the app in your browser:
   http://localhost/student_feedback/public/index.php

Login Accounts
- Use any login accounts you created in your users table.
